### Installing

1. 프로젝트 클론:
    ```sh
    git clone https://github.com/dongju0561/recordEvent.git
    ```
2. 프로젝트 디렉토리로 이동:
    ```sh
    cd recordEvent
    ```
3. 프로그램 빌드
    ```sh
    make
    ```
